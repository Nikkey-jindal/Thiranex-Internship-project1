# 🏥 MediCare Hospital Website

A modern, responsive, and accessible hospital website developed as **Task 2** during my **Frontend Development Internship at Thiranex**.

The project focuses on implementing **HTML5 Semantic Structure**, **Accessibility**, and **Responsive Web Design** using HTML5 and CSS3.

---

## 📌 Project Overview

MediCare Hospital is a responsive healthcare website designed to provide patients with an easy and user-friendly experience. The website showcases hospital services, medical departments, experienced doctors, and contact information while following modern web development best practices.

---

## ✨ Features

- Responsive Design
- Semantic HTML5 Structure
- Accessibility-Friendly Layout
- Modern Navigation Bar
- Hero Section
- About Us Section
- Medical Departments
- Expert Doctors Section
- Contact Form
- Professional Footer
- Smooth Scrolling
- Hover Animations
- Mobile, Tablet & Desktop Support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
